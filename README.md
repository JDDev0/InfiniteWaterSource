# Infinite Water Source

This Hytale mod makes water sources in a 2x2 and 1x3 layout infinite.

Waterlogged blocks are supported too.

Download on CurseForge: [Infinite Water Source](https://www.curseforge.com/hytale/mods/infinite-water-source)<br>
Download on Modtale: [Infinite Water Source](https://modtale.net/mod/infinite-water-source-504ac592-d5f3-407c-896b-fe4fb028bc09)
